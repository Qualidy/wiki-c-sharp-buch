    2.4 Verzweigungen mit if und else

    if/else – Temperaturklassifikation

    Lies eine Temperatur aus einem NumericUpDown (z. B. NumTemp) und gib im Label aus:

    • < 0 → „Frost“
    • == 0 → „Gefrierpunkt“
    • > 0 → „Plusgrade“
    Lösung 
    decimal t = NumTemp.Value;

if (t < 0)
{
    LblAnzeige.Text = "Frost";
}
else if (t == 0)
{
    LblAnzeige.Text = "Gefrierpunkt";
}
else
{
    LblAnzeige.Text = "Plusgrade";
}
    if/else – Notentext (1–6)

    Lies eine Note 1–6 (z. B. NumNote) und gib per if/else if/else den Text aus:

    • 1 „sehr gut“, 2 „gut“, 3 „befriedigend“, 4 „ausreichend“, 5 „mangelhaft“, 6 „ungenügend“.
    Lösung 
    int note = (int)NumNote.Value;
string text;

if (note == 1)
{
    text = "sehr gut";
}
else if (note == 2)
{
    text = "gut";
}
else if (note == 3)
{
    text = "befriedigend";
}
else if (note == 4)
{
    text = "ausreichend";
}
else if (note == 5)
{
    text = "mangelhaft";
}
else
{
    text = "ungenügend";
}

LblAnzeige.Text = text;
    Ternärer Operator – gerade/ungerade

    Lies eine Ganzzahl (NumericUpDown, z. B. NumZahl, DecimalPlaces=0) und gib „gerade“/„ungerade“ mit ?: aus.

    Tipp

    Mit hilfe des %-Operators kann man bestimmen, ob eine Zahl gerade oder ungerade ist. Rechachiere wie das geht!

    Lösung 
    int n = (int)NumZahl.Value;
string text = (n % 2 == 0) ? "gerade" : "ungerade";
LblAnzeige.Text = text;
    if/else – Wertebereiche

    Lies einen Wert −10…+10 (NumWert). Gib aus:

    • < 0 → „negativ“
    • == 0 → „null“
    • > 0 && <= 9 → „positiv, einstellig“
    • >= 10 → „positiv und mindestens zweistellig“
    Lösung 
    int v = (int)NumWert.Value;
string text;

if (v < 0)
{
    text = "negativ";
}
else if (v == 0)
{
    text = "null";
}
else if (v > 0 && v <= 9)
{
    text = "positiv, einstellig";
}
else
{
    text = "positiv und mindestens zweistellig";
}


LblAnzeige.Text = text;
