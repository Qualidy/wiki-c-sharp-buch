    2.1 Variablen und Datentypen

    Visitenkarte mit Variablen

    Lege Variablen für eine kleine Visitenkarte an und gib sie mehrzeilig aus.

    • string name
    • int alter
    • double groesseMeter
    • bool magCSharp

    Erzeuge eine mehrzeilige Zeichenkette mit String-Interpolation (z. B. via \n) und gib sie in der Oberfläche aus (z. B. Label oder MessageBox).

    Lösung 
    string name = "Alex";
int alter = 28;
double groesseMeter = 1.82;
bool magCSharp = true;

string text =
    $"Name: {name}\n" +
    $"Alter: {alter}\n" +
    $"Größe (m): {groesseMeter}\n" +
    $"Mag C#: {magCSharp}";

// WinForms-Beispiele:
// LblAnzeige.Text = text;
MessageBox.Show(text, "Visitenkarte");
    Passende Datentypen wählen und anzeigen

    Lege sinnvolle Variablen + Werte an und gib sie aus:

    • Kontostand (Geldbetrag)
    • Artikelanzahl (ganzzahlig)
    • Temperatur in °C (mit Nachkommastellen)
    • Ampelzustand (ein Buchstabe z. B. R/G/Y)

    Formatiere die Ausgabe ordentlich (mehrzeilig).

    Lösung 
    decimal kontostand = 1234.56m; // Geld: decimal
int artikelanzahl = 7;         // Stückzahl: int
double temperaturC = -2.5;     // Messwert mit Nachkommastellen: double
char ampel = 'R';              // ein Zeichen: char

string text =
    $"Kontostand: {kontostand} €\n" +
    $"Artikelanzahl: {artikelanzahl}\n" +
    $"Temperatur: {temperaturC} °C\n" +
    $"Ampel: {ampel}";

// LblAnzeige.Text = text;
MessageBox.Show(text, "Datentypen");
    Gültigkeitsbereich von Variablen (Feld vs. lokal)

    Zeige den Unterschied zwischen einem Feld (Klassenebene) und einer lokalen Variablen (Methodenebene):

    • Lege ein Feld globalZaehler in der Form-Klasse an.
    • In einem Button-Klick: Erhöhe globalZaehler und außerdem eine lokale Variable lokalerZaehler.
    • Gib beide Werte aus.
    • Erläutere im Kommentar, warum lokalerZaehler außerhalb der Methode nicht sichtbar ist.
    Lösung 
    // In Form1.cs (innerhalb der Klasse Form1, aber außerhalb der Methoden):
private int globalZaehler = 0;

// In der Klick-Methode (z. B. eines Buttons):
private void BtnKlick(object sender, EventArgs e)
{
    int lokalerZaehler = 0; // nur in dieser Methode sichtbar

    globalZaehler++;
    lokalerZaehler++;

    LblAnzeige.Text = $"global: {globalZaehler}, lokal: {lokalerZaehler}";
    // MessageBox.Show($"global: {globalZaehler}, lokal: {lokalerZaehler}");
}

// Hinweis:
// 'lokalerZaehler' existiert nur innerhalb von BtnKlick.
// Diese Zeile außerhalb der Methode wäre ein Fehler:
// LblAnzeige.Text = lokalerZaehler.ToString();
    Konstante und Preisberechnung (Brutto aus Netto)

    Lege eine Konstante für den Steuersatz an und berechne aus einem Netto-Wert den Brutto-Preis.

    Vorgaben: - const decimal STEUERSATZ = 0.19m - Beispiel: netto = 59.90m - Ausgabe: Netto, Steuerbetrag, Brutto — jeweils auf 2 Nachkommastellen.

    Lösung 
    const decimal STEUERSATZ = 0.19m;
decimal netto = 59.90m;

decimal steuer = netto * STEUERSATZ;
decimal brutto = netto + steuer;

string text =
    $"Netto:  {netto:F2} €\n" +
    $"Steuer: {steuer:F2} €\n" +
    $"Brutto: {brutto:F2} €";

// LblAnzeige.Text = text;
MessageBox.Show(text, "Preisberechnung");
    Typkonvertierung (implizit/explicit)

    Demonstriere einfache Typkonvertierungen:

    • int → double (implizit)
    • double → int (explizit per Cast)

    Gib beide Ergebnisse aus und notiere im Kommentar, was bei der Umwandlung double → int passiert.

    Lösung 
    int anzahl = 7;
double alsDouble = anzahl; // implizit möglich (kein Informationsverlust)

double preisProStueck = 2.75;
int ganzzahlig = (int)preisProStueck; // explizit: Nachkommastellen werden abgeschnitten

string text =
    $"anzahl (int): {anzahl}\n" +
    $"alsDouble (double): {alsDouble}\n" +
    $"preisProStueck (double): {preisProStueck}\n" +
    $"ganzzahlig (int, gecastet): {ganzzahlig}";

// LblAnzeige.Text = text;
MessageBox.Show(text, "Typkonvertierung");

    2.2 Operatoren

    Operatorrangfolge sichtbar machen

    Berechne die folgenden Ausdrücke und vergleiche die Ergebnisse.

    1) Ohne Klammern: 5 + 3 * 4 - 6 / 2 2) Mit Klammern (Variante A): (5 + 3) * 4 - 6 / 2 3) Mit Klammern (Variante B): 5 + (3 * 4 - 6) / 2

    Erkläre in kurzen Kommentaren, welche Operatoren zuerst ausgewertet werden.

    Lösung 
    int expr1 = 5 + 3 * 4 - 6 / 2;           // * und / vor + und -
int exprA = (5 + 3) * 4 - 6 / 2;         // Klammern zuerst
int exprB = 5 + (3 * 4 - 6) / 2;         // Innenklammer zuerst, dann /, dann +

string text =
    $"ohne Klammern: {expr1}\n" +
    $"Variante A   : {exprA}\n" +
    $"Variante B   : {exprB}";

// LblAnzeige.Text = text;
MessageBox.Show(text, "Operatorrangfolge");

    Lege Variablen an und gib die Ergebnisse der Vergleiche aus:

    • int a = 5, int b = 10, double x = 3.5
    • Prüfe: a < b, a * 3 == b, x >= 3.5, "Maier" == "Mayer"
    Lösung 
    int a = 5, b = 10;
double x = 3.5;

bool r1 = a < b;
bool r2 = a * 3 == b;
bool r3 = x >= 3.5;
bool r4 = "Maier" == "Mayer";

string text =
    $"a < b            : {r1}\n" +
    $"a * 3 == b       : {r2}\n" +
    $"x >= 3.5         : {r3}\n" +
    $"\"Maier\"==\"Mayer\": {r4}";

// LblAnzeige.Text = text;
MessageBox.Show(text, "Vergleichsoperatoren");
    Logische Operatoren (AND, OR, XOR, NOT)

    Verwende die Variablen a = 5, b = 10 und prüfe:

    • a > 0 && b != 10
    • a > 0 || b != 10
    • (a > 0) ^ (b != 10) (XOR)
    • !(a > 0)

    Notiere in 1–2 Kommentaren die Unterschiede zwischen &&, || und ^.

    Lösung 
    int a = 5, b = 10;

bool andRes = a > 0 && b != 10;     // true AND false -> false
bool orRes  = a > 0 || b != 10;     // true OR false  -> true
bool xorRes = (a > 0) ^ (b != 10);  // true XOR false -> true (genau ein Operand wahr)
bool notRes = !(a > 0);             // NOT true       -> false

string text =
    $"AND (&&): {andRes}\n" +
    $"OR  (||): {orRes}\n" +
    $"XOR  (^): {xorRes}\n" +
    $"NOT  (!): {notRes}";

// LblAnzeige.Text = text;
MessageBox.Show(text, "Logische Operatoren");
    Zusammengesetzte Zuweisungen

    Starte mit int n = 10 und führe nacheinander aus:

    • n += 5
    • n -= 3
    • n *= 2
    • n /= 4

    Gib nach jedem Schritt den aktuellen Wert aus.

    Lösung 
    int n = 10;

string text = $"Start: n = {n}\n";
n += 5;  text += $"nach n += 5 : {n}\n";
n -= 3;  text += $"nach n -= 3 : {n}\n";
n *= 2;  text += $"nach n *= 2 : {n}\n";
n /= 4;  text += $"nach n /= 4 : {n}\n";

// LblAnzeige.Text = text;
MessageBox.Show(text, "Zuweisungsoperatoren");
    Klammern bei booleschen Ausdrücken

    Lege boolesche Ausdrücke A, B, C fest, z. B.:

    • A = (5 < 0)
    • B = (10 % 2 == 0)
    • C = (3.5 >= 3)

    Vergleiche die Auswertung von - !A || B && C - (!A || B) && C

    Erkläre in einem Kommentar, warum die Ergebnisse unterschiedlich sein können.

    Lösung 
    bool A = (5 < 0);          // false
bool B = (10 % 2 == 0);    // true
bool C = (3.5 >= 3);       // true

bool expr1 = !A || B && C;       // && vor ||
bool expr2 = (!A || B) && C;     // Klammern ändern die Rangfolge

string text =
    $"!A || B && C     : {expr1}\n" +
    $"(!A || B) && C   : {expr2}\n" +
    $"Hinweis: '&&' hat höhere Priorität als '||'. Klammern können das ändern.";

// LblAnzeige.Text = text;
MessageBox.Show(text, "Boolesche Ausdrücke & Klammern");

    2.3 Einfache Steuerelemente

    Panel bewegen (Buttons)

    Bewege ein Panel mit vier Buttons um jeweils 10 Pixel:

    • CmdUp, CmdDown, CmdLeft, CmdRight
    • Ein Label LblAnzeige zeigt Position: X: <Left> Y: <Top>
    Lösung 
    // Ereignisse der vier Buttons (WinForms)
private void CmdUp_Click(object sender, EventArgs e)
{
    PnlBox.Top -= 10;
    LblAnzeige.Text = $"X: {PnlBox.Left}  Y: {PnlBox.Top}";
}
private void CmdDown_Click(object sender, EventArgs e)
{
    PnlBox.Top += 10;
    LblAnzeige.Text = $"X: {PnlBox.Left}  Y: {PnlBox.Top}";
}
private void CmdLeft_Click(object sender, EventArgs e)
{
    PnlBox.Left -= 10;
    LblAnzeige.Text = $"X: {PnlBox.Left}  Y: {PnlBox.Top}";
}
private void CmdRight_Click(object sender, EventArgs e)
{
    PnlBox.Left += 10;
    LblAnzeige.Text = $"X: {PnlBox.Left}  Y: {PnlBox.Top}";
}
    Timer-Automove (Wrap-around ohne if)

    Lasse ein Panel automatisch nach rechts laufen. Am rechten Rand soll es links wieder erscheinen (Wrap-around), ganz ohne if.

    • Timer (z. B. Intervall 30 ms), CmdStart, CmdStop, Panel PnlBox, Label LblAnzeige.
    Lösung 
    // Start/Stop
private void CmdStart_Click(object sender, EventArgs e)
{
    Tmr.Interval = 30;
    Tmr.Enabled = true;
}
private void CmdStop_Click(object sender, EventArgs e)
{
    Tmr.Enabled = false;
}

// Tick: Wrap-around per Modulo
private void Tmr_Tick(object sender, EventArgs e)
{
    int maxX = this.ClientSize.Width - PnlBox.Width;
    int next = PnlBox.Left + 3;
    PnlBox.Left = next % (maxX + 1);  // ohne if
    LblAnzeige.Text = $"X: {PnlBox.Left}";
}

    2.4 Verzweigungen mit if und else

    if/else – Temperaturklassifikation

    Lies eine Temperatur aus einem NumericUpDown (z. B. NumTemp) und gib im Label aus:

    • < 0 → „Frost“
    • == 0 → „Gefrierpunkt“
    • > 0 → „Plusgrade“
    Lösung 
    decimal t = NumTemp.Value;
string text;

if (t < 0)
    text = "Frost";
else if (t == 0)
    text = "Gefrierpunkt";
else
    text = "Plusgrade";

LblAnzeige.Text = text;
    if/else – Notentext (1–6)

    Lies eine Note 1–6 (z. B. NumNote) und gib per if/else if/else den Text aus:

    • 1 „sehr gut“, 2 „gut“, 3 „befriedigend“, 4 „ausreichend“, 5 „mangelhaft“, 6 „ungenügend“.
    Lösung 
    int note = (int)NumNote.Value;
string text;

if (note == 1) text = "sehr gut";
else if (note == 2) text = "gut";
else if (note == 3) text = "befriedigend";
else if (note == 4) text = "ausreichend";
else if (note == 5) text = "mangelhaft";
else text = "ungenügend";

LblAnzeige.Text = text;
    Ternärer Operator – gerade/ungerade

    Lies eine Ganzzahl (NumericUpDown, z. B. NumZahl, DecimalPlaces=0) und gib „gerade“/„ungerade“ mit ?: aus.

    Lösung 
    int n = (int)NumZahl.Value;
string text = (n % 2 == 0) ? "gerade" : "ungerade";
LblAnzeige.Text = text;
    if/else – Einlasskontrolle

    Lies Alter (NumAlter) und Ticket (ChkTicket). Gib aus:

    • Ab 18 und Ticket → „Einlass“
    • sonst „kein Einlass“
    Lösung 
    int alter = (int)NumAlter.Value;
bool ticket = ChkTicket.Checked;

if (alter >= 18 && ticket)
    LblAnzeige.Text = "Einlass";
else
    LblAnzeige.Text = "kein Einlass";
    if/else – Wertebereiche

    Lies einen Wert −10…+10 (NumWert). Gib aus:

    • < 0 → „negativ“
    • == 0 → „null“
    • > 0 && <= 9 → „positiv, einstellig“
    • >= 10 → „positiv, zweistellig“
    Lösung 
    int v = (int)NumWert.Value;
string text;

if (v < 0) text = "negativ";
else if (v == 0) text = "null";
else if (v > 0 && v <= 9) text = "positiv, einstellig";
else text = "positiv, zweistellig";

LblAnzeige.Text = text;

    2.5 Verzweigungen mit switch

    switch – Wochentag zu Werktag/Wochenende

    Lege enum Wochentag { Mo, Di, Mi, Do, Fr, Sa, So } und lies einen Wert (z. B. aus ComboBox). Gib mit switch aus: „Werktag“ für Mo–Fr, „Wochenende“ für Sa/So.

    Lösung 
    // enum (einmalig auf Klassenebene)
enum Wochentag { Mo, Di, Mi, Do, Fr, Sa, So }

// Annahme: ausgewählter Wert in 'tag'
Wochentag tag = Wochentag.Mo; // Beispiel

string text = tag switch
{
    Wochentag.Sa or Wochentag.So => "Wochenende",
    _ => "Werktag"
};

LblAnzeige.Text = text;
    switch mit Vergleichsmustern (NumericUpDown)

    Lies einen Double-Wert (NumericUpDown NumX). Nutze switch mit Vergleichsmustern:

    • < 0, 0, > 0 and <= 9, > 9
    Lösung 
    double x = (double)NumX.Value;

string text = x switch
{
    < 0 => "negativ",
    0 => "null",
    > 0 and <= 9 => "positiv, 0..9",
    > 9 => "positiv, > 9"
};

LblAnzeige.Text = text;
    switch-Ausdruck – Ampelfarbe

    Lies eine Zahl 1–3 (NumAmpel) und mappe per switch-Ausdruck auf „rot“, „gelb“, „grün“. Sonst „unbekannt“.

    Lösung 
    int code = (int)NumAmpel.Value;

string farbe = code switch
{
    1 => "rot",
    2 => "gelb",
    3 => "grün",
    _ => "unbekannt"
};

LblAnzeige.Text = farbe;
    switch – or-Muster (Städte)

    Lies einen Stadtnamen (TextBox TxtStadt). Für „Namur“, „Antwerpen“ oder „Lüttich“ gib „Belgien (BE)“ aus; sonst „andere“.

    Lösung 
    string s = TxtStadt.Text.Trim();

string land = s switch
{
    "Namur" or "Antwerpen" or "Lüttich" => "Belgien (BE)",
    _ => "andere"
};

LblAnzeige.Text = land;
    switch – Tupel & when

    Lies zwei Ganzzahlen (z. B. NumA, NumB) und verwende einen Tupel-switch mit when:

    • beide < 10
    • beide > 9
    • Spezialfall: a == b (per when)
    Lösung 
    int a = (int)NumA.Value;
int b = (int)NumB.Value;

string text = (a, b) switch
{
    var t when t.a == t.b => "gleich",
    var t when t.a < 10 && t.b < 10 => "beide < 10",
    var t when t.a > 9 && t.b > 9 => "beide > 9",
    _ => "gemischt"
};

LblAnzeige.Text = text;

    Gemeinsamer Handler für Schriftgröße

    Vier RadioButtons (8/10/12/16 pt) teilen sich eine Methode; diese liest ((RadioButton)sender).Text aus und setzt LblText.Font.

    Lösung 
    private void RbtSize_CheckedChanged(object sender, EventArgs e)
{
    var r = sender as RadioButton;
    if (r != null && r.Checked)
    {
        if (float.TryParse(r.Text.Replace("pt","").Trim(), out float size))
        {
            LblText.Font = new Font(LblText.Font.FontFamily, size, LblText.Font.Style);
        }
    }
}
    GroupBox – zwei unabhängige Optionen

    Zwei GroupBoxen: A (Farbe: Rot/Grün/Blau), B (Stil: Fett/Kursiv/Unterstrichen).

    Wähle unabhängig – setze LblText.ForeColor und LblText.Font entsprechend.

    Lösung 
    private void Farbe_CheckedChanged(object sender, EventArgs e)
{
    if (!((RadioButton)sender).Checked) return;
    if (RbtRot.Checked)   LblText.ForeColor = Color.Red;
    if (RbtGruen.Checked) LblText.ForeColor = Color.Green;
    if (RbtBlau.Checked)  LblText.ForeColor = Color.Blue;
}

private void Stil_CheckedChanged(object sender, EventArgs e)
{
    if (!((RadioButton)sender).Checked) return;
    FontStyle style = FontStyle.Regular;
    if (RbtFett.Checked)       style |= FontStyle.Bold;
    if (RbtKursiv.Checked)     style |= FontStyle.Italic;
    if (RbtUnterstrichen.Checked) style |= FontStyle.Underline;
    LblText.Font = new Font(LblText.Font, style);
}
    TrackBar steuert Schriftgröße

    2.7 Schleifen

    for – Summe 1..n

    Lies n (NumericUpDown NumN) und berechne die Summe 1+…+n mit for. Gib Ergebnis aus.

    Lösung 
    int n = (int)NumN.Value;
int sum = 0;
for (int i = 1; i <= n; i++) sum += i;
LblAnzeige.Text = $"Summe 1..{n} = {sum}";
    for – Fakultät n!

    Lies n (klein halten, z. B. ≤ 10) und berechne n! mit for. Gib zusätzlich den Rechenweg aus.

    Lösung 
    int n = (int)NumN.Value;
long prod = 1;
string weg = "";
for (int i = 1; i <= n; i++)
{
    prod *= i;
    weg += (i == 1 ? "1" : $"×{i}");
}
LblAnzeige.Text = $"{weg} = {prod}";
    while – Countdown

    Starte bei n und zähle bis 0 herunter. Gib jede Zahl in einer neuen Zeile aus.

    Lösung 
    int n = (int)NumN.Value;
string text = "";
while (n >= 0)
{
    text += n + "\n";
    n--;
}
LblAnzeige.Text = text;
    while – Ziffern zählen

    Zähle die Dezimalstellen einer positiven Ganzzahl z durch wiederholtes Teilen durch 10.

    Lösung 
    int z = Math.Abs((int)NumZ.Value);
int count = 0;
do
{
    count++;
    z /= 10;
} while (z > 0);
LblAnzeige.Text = $"Ziffern: {count}";
    do…while – Verdoppeln bis Ziel

    Starte bei prod = 1 und verdopple (prod *= 2), bis prod >= n erreicht ist. Gib prod und die Anzahl der Verdopplungen aus.

    Lösung 
    int n = (int)NumN.Value;
int prod = 1;
int steps = 0;
do
{
    if (prod >= n) break;
    prod *= 2;
    steps++;
} while (true);
LblAnzeige.Text = $"prod={prod}, Schritte={steps}";

    2.8 Schleifen mit Steuerelementen

    ListBox füllen (1..n)

    Fülle eine ListBox (LstZahlen) mit den Zahlen 1..n (NumericUpDown NumN) per for.

    Lösung 
    LstZahlen.Items.Clear();
int n = (int)NumN.Value;
for (int i = 1; i <= n; i++) LstZahlen.Items.Add(i);
    Auswahl anzeigen

    Bei SelectedIndexChanged gib den markierten Eintrag und den Index im Label aus.

    Lösung 
    private void LstZahlen_SelectedIndexChanged(object sender, EventArgs e)
{
    if (LstZahlen.SelectedIndex >= 0)
    {
        var val = LstZahlen.SelectedItem;
        LblAnzeige.Text = $"Wert: {val}, Index: {LstZahlen.SelectedIndex}";
    }
}
    Summe aller Einträge (foreach)

    Bilde die Summe aller ListBox.Items (ganzzahlig). Nutze foreach.

    Lösung 
    int sum = 0;
foreach (var item in LstZahlen.Items)
    sum += Convert.ToInt32(item);
LblAnzeige.Text = $"Summe: {sum}";
    Summe der markierten Einträge

    Aktiviere Mehrfachauswahl (SelectionMode=MultiSimple oder MultiExtended) und summiere nur die ausgewählten Einträge.

    Lösung 
    int sum = 0;
foreach (var item in LstZahlen.SelectedItems)
    sum += Convert.ToInt32(item);
LblAnzeige.Text = $"Summe (selektiert): {sum}";
    ComboBox aus Array + Auswahl

    Erzeuge ein string[] mit fünf Städten und fülle die ComboBox (CmbStadt). Bei Auswahl: Stadt und Index anzeigen.

    Lösung 
    // Befüllen (z. B. im Form_Load)
string[] staedte = { "Kiel", "Hamburg", "Lübeck", "Flensburg", "Neumünster" };
CmbStadt.Items.Clear();
CmbStadt.Items.AddRange(staedte);

// Auswahl-Event
private void CmbStadt_SelectedIndexChanged(object sender, EventArgs e)
{
    if (CmbStadt.SelectedIndex >= 0)
    {
        LblAnzeige.Text = $"Stadt: {CmbStadt.SelectedItem}, Index: {CmbStadt.SelectedIndex}";
    }
