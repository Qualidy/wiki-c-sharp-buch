    Klasse Person – Eigenschaften, Konstruktor, ToString

    Implementiere eine Klasse Person mit

    • auto‑Properties Name (string) und Geburtsjahr (int)
    • Konstruktor (string name, int jahr)
    • override string ToString() → "Name (Jahr)"

    Erzeuge in der UI zwei Personen und gib sie jeweils aus.

    Lösung 
    public class Person
{
    public string Name { get; set; }
    public int Geburtsjahr { get; set; }

    public Person(string name, int jahr)
    {
        Name = name;
        Geburtsjahr = jahr;
    }

    public override string ToString() => $"{Name} ({Geburtsjahr})";
}

private void CmdPerson_Click(object sender, EventArgs e)
{
    var p1 = new Person("Alex", 1990);
    var p2 = new Person("Sam", 1985);
    LblAnzeige.Text = $"{p1}\n{p2}";
}
    Equals/GetHashCode – Personen vergleichen

    Erweitere Person, sodass zwei Personen als gleich gelten, wenn Name und Geburtsjahr übereinstimmen. Überschreibe Equals und GetHashCode passend und demonstriere den Vergleich.

    Lösung 
    public class Person
{
    public string Name { get; set; }
    public int Geburtsjahr { get; set; }

    public Person(string name, int jahr) { Name = name; Geburtsjahr = jahr; }

    public override string ToString() => $"{Name} ({Geburtsjahr})";

    public override bool Equals(object? obj)
    {
        return obj is Person p && p.Name == Name && p.Geburtsjahr == Geburtsjahr;
    }

    public override int GetHashCode() => HashCode.Combine(Name, Geburtsjahr);
}

private void CmdEq_Click(object sender, EventArgs e)
{
    var a = new Person("Alex", 1990);
    var b = new Person("Alex", 1990);
    MessageBox.Show($"a.Equals(b): {a.Equals(b)}");
}
    Vererbung – Mitarbeiter : Person mit virtual/override

    Leite Mitarbeiter von Person ab und ergänze eine Eigenschaft Abteilung. Überschreibe ToString() und rufe base.ToString() auf.

    Lösung 
    public class Mitarbeiter : Person
{
    public string Abteilung { get; set; }
    public Mitarbeiter(string name, int jahr, string abteilung) : base(name, jahr)
    {
        Abteilung = abteilung;
    }
    public override string ToString() => base.ToString() + $" – {Abteilung}";
}
    Abstrakte Klasse & Polymorphie

    Erstelle abstract class Figur mit abstract double Flaeche() und leite Rechteck und Kreis ab. Erzeuge beide und gib die Flächen über eine gemeinsame List<Figur> aus.

    Lösung 
    public abstract class Figur
{
    public abstract double Flaeche();
}
public class Rechteck : Figur
{
    public double W, H;
    public Rechteck(double w, double h) { W = w; H = h; }
    public override double Flaeche() => W * H;
}
public class Kreis : Figur
{
    public double R;
    public Kreis(double r) { R = r; }
    public override double Flaeche() => Math.PI * R * R;
}

private void CmdFiguren_Click(object sender, EventArgs e)
{
    var list = new List<Figur> { new Rechteck(3,2), new Kreis(2.5) };
    LblAnzeige.Text = string.Join("\n", list.Select(f => f.Flaeche().ToString("F2")));
}
    Interface – ISpeicherbar mit Speichern()

    Definiere interface ISpeicherbar { string Speichern(); } und implementiere es in Person. Die Methode liefert eine CSV‑Zeile zurück.

    Lösung 
    public interface ISpeicherbar { string Speichern(); }

public class Person : ISpeicherbar
{
    public string Name { get; set; }
    public int Geburtsjahr { get; set; }
    public Person(string n, int j) { Name = n; Geburtsjahr = j; }
    public string Speichern() => $"{Name};{Geburtsjahr}";
}
    Generics – List<Person> filtern & sortieren

    Lege eine List<Person> an, filtere alle mit Geburtsjahr ≥ 1990 und sortiere aufsteigend nach Name. Gib die Namen jeweils in einer neuen Zeile aus.

    Lösung 
    var list = new List<Person>
{
    new Person("Zoe", 1980),
    new Person("Alex", 1992),
    new Person("Mara", 1995)
};

var q = list.Where(p => p.Geburtsjahr >= 1990)
            .OrderBy(p => p.Name)
            .Select(p => p.Name);

LblAnzeige.Text = string.Join("\n", q);

    Operatorüberladung – Vektor addieren

    Erstelle einen struct Vektor2D mit X, Y und überlade + zur komponentenweisen Addition. Demonstriere die Addition in der UI.

    Lösung 
    public struct Vektor2D
{
    public double X, Y;
    public Vektor2D(double x, double y) { X = x; Y = y; }
    public static Vektor2D operator +(Vektor2D a, Vektor2D b) => new Vektor2D(a.X + b.X, a.Y + b.Y);
    public override string ToString() => $"({X},{Y})";
}

    Mehrere Formulare – Dialog für Person eingeben

    Erstelle ein zweites Formular FrmPersonEdit, in dem Name und Geburtsjahr eingegeben werden. Öffne es modal (ShowDialog) und übernimm die Daten zurück ins Hauptformular.

    Hinweis

    • Übergabe über öffentliche Properties des Dialogs.
    Lösung 
    // In FrmPersonEdit
public string PersonName { get; set; } = "";
public int Geburtsjahr { get; set; } = 2000;
// OK-Button: Werte aus Controls in Properties schreiben; DialogResult = OK;

// Im Hauptformular:
private void CmdEdit_Click(object sender, EventArgs e)
{
    using var dlg = new FrmPersonEdit();
    if (dlg.ShowDialog() == DialogResult.OK)
    {
        LblAnzeige.Text = $"{dlg.PersonName} ({dlg.Geburtsjahr})";
    }
}
