    string‑APIs – Trimmen, Ersetzen, Splitten

    Lies einen Text aus TxtRaw ein und bereite ihn auf:

    • Trim() führende/folgende Leerzeichen entfernen
    • Replace(" ", " ") doppelte Spaces vereinfachen
    • Split(' ') in Wörter zerlegen und Anzahl zählen

    Zeige Ergebnistext und Wortanzahl an.

    Lösung 
    string raw = TxtRaw.Text;
string cleaned = raw.Trim();
while (cleaned.Contains("  ")) cleaned = cleaned.Replace("  ", " ");

var words = cleaned.Split(' ', StringSplitOptions.RemoveEmptyEntries);
LblAnzeige.Text = cleaned + "\n" + $"Wörter: {words.Length}";
    DateTime & TimeSpan – Fälligkeit berechnen

    Lies ein Startdatum (DateTimePicker DtpStart) und eine Frist in Tagen (NumTage) ein.

    • Berechne das Fälligkeitsdatum: start + tage
    • Zeige an, wie viele Tage ab heute noch verbleiben (negativ, wenn überfällig).
    Lösung 
    DateTime start = DtpStart.Value.Date;
int tage = (int)NumTage.Value;
DateTime faellig = start.AddDays(tage);

int diff = (faellig - DateTime.Today).Days;
LblAnzeige.Text = $"Fällig am: {faellig:d}\nResttage: {diff}";
    Dateien – Text speichern & laden

    Implementiere Speichern und Laden für einen Text aus TxtInhalt in eine Datei notiz.txt im Programmverzeichnis.

    • File.WriteAllText
    • File.ReadAllText

    Fehler (z. B. fehlende Rechte) sollen als Meldung erscheinen.

    Lösung 
    using System.IO;

private void CmdSave_Click(object sender, EventArgs e)
{
    try { File.WriteAllText("notiz.txt", TxtInhalt.Text); }
    catch (Exception ex) { MessageBox.Show(ex.Message, "Speicherfehler"); }
}

private void CmdLoad_Click(object sender, EventArgs e)
{
    try { TxtInhalt.Text = File.ReadAllText("notiz.txt"); }
    catch (Exception ex) { MessageBox.Show(ex.Message, "Ladefehler"); }
}
    CSV – einfache Liste importieren/exportieren

    Speichere eine Liste von Personen (Name;Jahr) als CSV‑Datei und lade sie wieder.

    • Beim Speichern: jede Person in eine Zeile schreiben
    • Beim Laden: Zeilen splitten und Personenliste in einer ListBox anzeigen
    Lösung 
    using System.IO;

private List<Person> personen = new();

private void CmdExport_Click(object sender, EventArgs e)
{
    File.WriteAllLines("personen.csv", personen.Select(p => $"{p.Name};{p.Geburtsjahr}"));
}

private void CmdImport_Click(object sender, EventArgs e)
{
    personen = File.ReadAllLines("personen.csv")
                   .Select(l => l.Split(';'))
                   .Select(a => new Person(a[0], int.Parse(a[1])))
                   .ToList();
    Lst.Items.Clear();
    Lst.Items.AddRange(personen.Select(p => p.ToString()).ToArray());
}
    XML – einfache Struktur speichern/laden

    Speichere eine kleine Konfiguration als XML und lade sie wieder.

    Beispielstruktur

    • <config>
    • <window title="..." width="..." height="..."/>
    • </config>
    Lösung 
    using System.Xml.Linq;

private void CmdXmlSave_Click(object sender, EventArgs e)
{
    var doc = new XDocument(
        new XElement("config",
            new XElement("window",
                new XAttribute("title", this.Text),
                new XAttribute("width", this.Width),
                new XAttribute("height", this.Height)
            )
        )
    );
    doc.Save("config.xml");
}

private void CmdXmlLoad_Click(object sender, EventArgs e)
{
    var doc = XDocument.Load("config.xml");
    var win = doc.Root!.Element("window")!;
    this.Text = (string)win.Attribute("title")!;
    this.Width = (int)win.Attribute("width")!;
    this.Height = (int)win.Attribute("height")!;
}
    Math – Funktionswerte in ListBox

    Berechne für x = -3 … +3 in 0,5‑Schritten die Werte f(x) = sin(x) + cos(x) und zeige sie formatiert in einer ListBox.

    Lösung 
    Lst.Items.Clear();
for (double x = -3; x <= 3.001; x += 0.5)
{
    double y = Math.Sin(x) + Math.Cos(x);
    Lst.Items.Add($"x={x,4:F1}  f(x)={y,7:F3}");
}
    DateTimePicker – Zeitraumdiff in Stunden/Minuten

    Nutze zwei DateTimePicker (DtpVon, DtpBis) und berechne Differenzen:

    • Gesamtdauer in Tagen
    • in Stunden
    • in Minuten
    Lösung 
    DateTime von = DtpVon.Value;
DateTime bis = DtpBis.Value;
var span = bis - von;

LblAnzeige.Text = $"Tage: {span.TotalDays:F2}\nStunden: {span.TotalHours:F2}\nMinuten: {span.TotalMinutes:F0}";
    Verzeichnisse – Liste der Dateien anzeigen

    Zeige alle Dateien eines Ordners in einer ListBox an. Nutze FolderBrowserDialog zur Auswahl.

    Lösung 
    using System.IO;

private void CmdPickFolder_Click(object sender, EventArgs e)
{
    using var dlg = new FolderBrowserDialog();
    if (dlg.ShowDialog() == DialogResult.OK)
    {
        Lst.Items.Clear();
        foreach (var f in Directory.GetFiles(dlg.SelectedPath))
            Lst.Items.Add(f);
    }
}
