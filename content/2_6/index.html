    2.6 Verzweigungen und Steuerelemente

    2.6.1 Steuerelement Checkbox

    Eine Checkbox

    Erstelle eine Checkbox mit den Text "Zustimmen". Erstelle ein Label mit dem schwarzen Text: "Wir benötigen ihre Zustimmung".

    Finde heraus, wie du die Farbe eines Textes ändern kannst.

    Sorge dafür, dass wenn du die Checkbox aktivierst, der Text grün wird und lautet "Danke für Ihre Zustimmung!".

    Wenn die Checkbox deaktiviert wird, dann soll wieder der ursprüngliche schwarze Text erscheinen.

    Bonus: Speichere die Farben und Texte in Konstanten.

    Tipp

    Textfarbe ändern:

    LblHinweis.ForeColor = Color.Green; // oder Color.Black
    Lösung 
    namespace DeineApp
{
    public partial class Form1 : Form
    {
        public Form1()
        {
            InitializeComponent();

            // Anfangszustand sicherstellen
            LblHinweis.Text = "Wir benötigen ihre Zustimmung";
            LblHinweis.ForeColor = Color.Black;

        }

        private void ChkZustimmen_CheckedChanged(object sender, EventArgs e)
        {
            if (ChkZustimmen.Checked)
            {
                LblHinweis.Text = "Danke für Ihre Zustimmung!";
                LblHinweis.ForeColor = Color.Green;
            }
            else
            {
                LblHinweis.Text = "Wir benötigen ihre Zustimmung";
                LblHinweis.ForeColor = Color.Black;
            }
        }
    }
}

    Lösung mit Bonus:

    namespace DeineApp
{
    public partial class Form1 : Form
    {
        // Texte als Konstanten
        private const string TEXT_DEFAULT = "Wir benötigen ihre Zustimmung";
        private const string TEXT_OK      = "Danke für Ihre Zustimmung!";

        // Farben als schreibgeschützte Felder
        private static readonly Color COLOR_DEFAULT = Color.Black;
        private static readonly Color COLOR_OK      = Color.Green;

        public Form1()
        {
            InitializeComponent();

            // Anfangszustand
            LblHinweis.Text = TEXT_DEFAULT;
            LblHinweis.ForeColor = COLOR_DEFAULT;
        }

        private void ChkZustimmen_CheckedChanged(object sender, EventArgs e)
        {
            if (ChkZustimmen.Checked)
            {
                LblHinweis.Text = TEXT_OK;
                LblHinweis.ForeColor = COLOR_OK;
            }
            else
            {
                LblHinweis.Text = TEXT_DEFAULT;
                LblHinweis.ForeColor = COLOR_DEFAULT;
            }
        }
    }
}
    Zwei Checkboxen, zwei Buttons

    Erstelle zwei Checkboxen und zwei Buttons.

    Sorge dafür, dass der erste Button nur dann klickbar ist, wenn mindestens einer der beiden Checkboxen einen Haken hat.

    Sorge weiter dafür, dass der zweite Button nur dann klickbar ist, wenn beide Checkboxen einen Haken haben.

    Lösung

    WinForms (VS 2022)

    Benennung im Designer
    Setze im Eigenschaftenfenster die Namen:

    • ChkLinks, ChkRechts (CheckBox)
    • CmdErster, CmdZweiter (Button)
    namespace DeineApp
{
    public partial class Form1 : Form
    {
        public Form1()
        {
            InitializeComponent();
        }

        private void ChkLinks_CheckedChanged(object sender, EventArgs e)
        {
            CmdErster.Enabled = ChkLinks.Checked || ChkRechts.Checked;
            CmdZweiter.Enabled = ChkLinks.Checked && ChkRechts.Checked;
        }

        private void ChkRechts_CheckedChanged(object sender, EventArgs e)
        {
            CmdErster.Enabled = ChkLinks.Checked || ChkRechts.Checked;
            CmdZweiter.Enabled = ChkLinks.Checked && ChkRechts.Checked;
        }

        private void CmdErster_Click(object sender, EventArgs e)
        {
            MessageBox.Show("Erster Button geklickt.");
        }

        private void CmdZweiter_Click(object sender, EventArgs e)
        {
            MessageBox.Show("Zweiter Button geklickt.");
        }
    }
}
    Versteckter Knopf

    Erstelle zwei Checkboxen.

    Sorge dafür, dass man zu Beginn nur eine von den zwei Boxen nutzen kann.

    Sobald die erste Checkbox aktiviert wurde, kann auch die andere Checkbox angeklickt werden.

    Wenn die zweite Checkbox aktiviert wurde, dann erscheint ein bisher unsichtbarer Button. Wenn auf diesen Button geklickt wird, wird das Programm geschlossen.

    Lösung

    WinForms (VS 2022)

    Benennung im Designer
    - CheckBox 1: ChkErste (Text z. B. Erste Option)
    - CheckBox 2: ChkZweite (Text z. B. Zweite Option)
    - Button: CmdBeenden (Text: Beenden)

    Startzustand (im Designer setzen oder im Code unten):
    - ChkZweite.Enabled = false
    - CmdBeenden.Visible = false

    Form1.cs 

    namespace DeineApp
{
    public partial class Form1 : Form
    {
        public Form1()
        {
            InitializeComponent();
        }

        // Sobald die erste Checkbox AKTIV ist, darf die zweite benutzt werden.
        private void ChkErste_CheckedChanged(object sender, EventArgs e)
        {
            ChkZweite.Enabled = ChkErste.Checked;
            if (!ChkZweite.Enabled)
            {
                // Wenn wieder gesperrt, zweite abwählen und Button verstecken
                ChkZweite.Checked = false;
                CmdBeenden.Visible = false;
            }
        }

        // Wenn die zweite Checkbox aktiviert wird, erscheint der Button.
        // (Deaktivieren blendet ihn wieder aus.)
        private void ChkZweite_CheckedChanged(object sender, EventArgs e)
        {
            CmdBeenden.Visible = ChkZweite.Checked;
        }

        // Button schließt die Anwendung.
        private void CmdBeenden_Click(object sender, EventArgs e)
        {
            Close();
        }
    }
}

    Vier Checkboxen – „Alle auswählen“

    Erstelle vier Checkboxen. Die erste Checkbox erhält den Text „Alle auswählen“.

    Programmiere die erste Checkbox so, dass wenn sie aktiviert wird, auch alle anderen Checkboxen aktiviert werden (und beim Deaktivieren alle anderen wieder deaktiviert werden).

    Lösung

    WinForms (VS 2022)

    Benennung im Designer
    - CheckBox 1: ChkAlleAuswaehlen (Text: Alle auswählen)
    - CheckBox 2: ChkOption1 (Text z. B. Option 1)
    - CheckBox 3: ChkOption2 (Text z. B. Option 2)
    - CheckBox 4: ChkOption3 (Text z. B. Option 3)

    Form1.cs 

    namespace DeineApp
{
    public partial class Form1 : Form
    {
        public Form1()
        {
            InitializeComponent();

            // Anfangszustand (optional)
            ChkAlleAuswaehlen.Checked = false;
            ChkOption1.Checked = false;
            ChkOption2.Checked = false;
            ChkOption3.Checked = false;
        }

        // "Alle auswählen" steuert die übrigen Checkboxen
        private void ChkAlleAuswaehlen_CheckedChanged(object sender, EventArgs e)
        {
            bool state = ChkAlleAuswaehlen.Checked;
            ChkOption1.Checked = state;
            ChkOption2.Checked = state;
            ChkOption3.Checked = state;
        }
    }
}

    2.6.2 Steuerlement Radiobutton

    3 Radio Buttons - 3 Funktionen

    Erstelle drei Radiobuttons und ein Label. Sorge dafür, dass sich die Farbe des Textes ändert, wenn man einen der Radiobuttons klickt.

    Baue die Lösung dabei so, dass jeder Button eine eigene Funktion hat.

    Lösung 
    // Angenommene Namen:
//   RadioButtons: RbRot, RbGruen, RbBlau
//   Label:        LblText
// Jeder RadioButton hat seine eigene Ereignisfunktion (CheckedChanged).

private void RbRot_CheckedChanged(object sender, EventArgs e)
{
    if (RbRot.Checked) LblText.ForeColor = Color.Red;
}

private void RbGruen_CheckedChanged(object sender, EventArgs e)
{
    if (RbGruen.Checked) LblText.ForeColor = Color.Green;
}

private void RbBlau_CheckedChanged(object sender, EventArgs e)
{
    if (RbBlau.Checked) LblText.ForeColor = Color.Blue;
}
    3 Radio Buttons deaktivieren

    Erstelle zwei Radiobuttons.

    Erstelle einen Button, der dafür sorgt, dass keiner der zwei Radiobuttons ausgewählt ist.

    Erstelle einen weiteren Button. Konfiguriere ihn so, dass er nur dann anklickbar ist, einer der Radiobuttons ausgewählt ist.

    Lösung 
    // Angenommene Namen:
//   RadioButtons: RbA, RbB
//   Button zum Leeren: BtnLeeren
//   Nur-aktiv-wenn-ausgewählt-Button: BtnAktion
// Hinweis: Setze im Designer anfangs BtnAktion.Enabled = false.

private void RbA_CheckedChanged(object sender, EventArgs e)
{
    // Aktivierbar nur, wenn mind. einer ausgewählt ist.
    if (RbA.Checked || RbB.Checked)
    {
        BtnAktion.Enabled = true;
    }
    else
    {
        BtnAktion.Enabled = false;
    }
}

private void RbB_CheckedChanged(object sender, EventArgs e)
{
    // Gleiche Logik wie oben (kein Extra-Helfer, wie gefordert bis hierhin).
    if (RbA.Checked || RbB.Checked)
    {
        BtnAktion.Enabled = true;
    }
    else
    {
        BtnAktion.Enabled = false;
    }
}

private void BtnLeeren_Click(object sender, EventArgs e)
{
    // Beide Radiobuttons abwählen -> nichts ausgewählt.
    RbA.Checked = false;
    RbB.Checked = false;

    // Konsistent: Aktion-Button ist nun nicht anklickbar.
    BtnAktion.Enabled = false;
}
    Textfeld aktivieren

    Erstelle drei Radiobuttons mit den Texten "Herr", "Frau", "Frei wählbar".

    Erstellen Sie eine Textbox neben den dritten Radiobutton. Konfiguriere diesen so, dass man nur in diesen schreiben kann, wenn "Frei wählbar" ausgewählt ist.

    Erstelle eine weitere Textbox für deinen Namen.

    Erstelle einen Button mit dem Text "Grüß mich", der Nutzer mit seinem Namen und der gewünschten Anrede anspricht.

    Lösung 
    // Angenommene Namen:
//   RbHerr, RbFrau, RbFrei  (alle -> CheckedChanged = RbAnrede_CheckedChanged)
//   TbFrei (Enabled = false), TbName, BtnGruesMich (-> Click), LblGruss

private void RbAnrede_CheckedChanged(object sender, EventArgs e)
{
    TbFrei.Enabled = RbFrei.Checked;
}

private void BtnGruesMich_Click(object sender, EventArgs e)
{
    string anrede = "Herr";
    if (RbFrau.Checked) { anrede = "Frau"; }
    if (RbFrei.Checked) { anrede = TbFrei.Text; }

    LblGruss.Text = $"Hallo {anrede} {TbName.Text}!";
}

    2.6.3 Gemeinsame Methode für mehrere Ereignisse

    3 Radio Buttons - 1 Funktion

    Erstelle drei Radiobuttons und ein Label. Sorge dafür, dass sich die Farbe des Textes ändert, wenn man einen der Radiobuttons klickt.

    Baue die Lösung dabei so, dass alle Buttons die gleiche Funktion benutzen .

    Lösung 
    // Angenommene Namen:
//   RadioButtons: RbRot, RbGruen, RbBlau  (alle -> CheckedChanged = RbFarbe_CheckedChanged)
//   Label:        LblText

private void RbFarbe_CheckedChanged(object sender, EventArgs e)
{
    // Ein und dieselbe Funktion für alle drei RadioButtons.
    RadioButton rb = (RadioButton)sender;

    if (!rb.Checked)
    {
        return;
    }

    if (rb == RbRot)
    {
        LblText.ForeColor = Color.Red;
    }
    else if (rb == RbGruen)
    {
        LblText.ForeColor = Color.Green;
    }
    else if (rb == RbBlau)
    {
        LblText.ForeColor = Color.Blue;
    }
}

    2.6.4 Steuerelement GroupBox

    2 GroupBoxen – Kontakt & Versand

    Erstelle ein kleines Formular mit zwei GroupBoxen.

    1) GroupBox „Kontakt“ (GbxKontakt)

    • Zwei RadioButtons: RbMail (Text: „E-Mail“) und RbTelefon (Text: „Telefon“).

    2) GroupBox „Versand“ (GbxVersand)

    • Zwei RadioButtons: RbStandard (Text: „Standard“) und RbExpress (Text: „Express“).

    3) Auswertung

    • Ein Button BtnAuswerten (Text: „Auswahl anzeigen“).

    • Ein Label LblAusgabe (leer).

    • Beim Klick auf den Button soll im Label stehen, welche Kontaktart und welche Versandart gewählt wurden, z. B.: Kontakt: E-Mail | Versand: Express.

    • Starte ohne Vorauswahl.

    • Der Button BtnAuswerten ist nur anklickbar, wenn in beiden GroupBoxen jeweils eine Option gewählt wurde.

    Lösung 
    // Angenommene Namen:
//   GroupBoxen: GbxKontakt, GbxVersand
//   RadioButtons: RbMail, RbTelefon, RbStandard, RbExpress
//   Button: BtnAuswerten
//   Label: LblAusgabe
// Hinweise: Setze im Designer anfangs BtnAuswerten.Enabled = false.
//           Starte ohne Vorauswahl (alle RadioButtons: Checked = false).

// Allen vier RadioButtons im Designer dasselbe Ereignis zuweisen:
// CheckedChanged = Rb_CheckedChanged

private void Rb_CheckedChanged(object sender, EventArgs e)
{
    bool kontaktGewählt = RbMail.Checked || RbTelefon.Checked;
    bool versandGewählt = RbStandard.Checked || RbExpress.Checked;
    BtnAuswerten.Enabled = kontaktGewählt && versandGewählt;
}

private void BtnAuswerten_Click(object sender, EventArgs e)
{
    // Button ist nur aktiv, wenn in beiden Gruppen etwas gewählt ist.
    string kontakt = RbMail.Checked ? "E-Mail" : "Telefon";
    string versand = RbExpress.Checked ? "Express" : "Standard";
    LblAusgabe.Text = $"Kontakt: {kontakt} | Versand: {versand}";
}
    Checkbox zeigt GroupBox mit 2 Optionen

    Erstelle eine CheckBox ChkZusatz (Text: „Zusatzoptionen anzeigen“).

    Erstelle eine GroupBox GbxZusatz (Text: „Zusatzoptionen“) mit zwei RadioButtons:

    • RbA (Text: „Option A“) – soll vorausgewählt sein.

    • RbB (Text: „Option B“).

    Erstelle ein immer sichtbares Label LblAuswahl, das die aktuelle Auswahl beschreibt.

    Starte so:

    • GbxZusatz.Visible = false

    • RbA.Checked = true (Vorauswahl)

    • LblAuswahl.Text = "Auswahl: (deaktiviert)"

    Verhalten:

    • Wenn ChkZusatz angehakt wird → GbxZusatz sichtbar und LblAuswahl zeigt die aktuell ausgewählte Option.

    • Wenn ChkZusatz abgewählt wird → GbxZusatz unsichtbar und LblAuswahl zeigt „(deaktiviert)“.

    • Beim Umschalten zwischen RbA und RbB wird LblAuswahl entsprechend aktualisiert.

    Lösung 
    // Angenommene Namen und Verdrahtung:
//   CheckBox:   ChkZusatz          (CheckedChanged = ChkZusatz_CheckedChanged)
//   GroupBox:   GbxZusatz          (Visible = false)
//   RadioBtns:  RbA, RbB           (beide CheckedChanged = RbOption_CheckedChanged)
//   Label:      LblAuswahl         (Text = "Auswahl: (deaktiviert)")
//   Designer:   RbA.Checked = true (Vorauswahl)

private void ChkZusatz_CheckedChanged(object sender, EventArgs e)
{
    GbxZusatz.Visible = ChkZusatz.Checked;

    if (ChkZusatz.Checked)
    {
        // Zeige die aktuell gewählte Option (RbA ist standardmäßig vorausgewählt)
        LblAuswahl.Text = $"Auswahl: {(RbA.Checked ? RbA.Text : RbB.Text)}";
    }
    else
    {
        LblAuswahl.Text = "Auswahl: (deaktiviert)";
    }
}

private void RbOption_CheckedChanged(object sender, EventArgs e)
{
    RadioButton rb = (RadioButton)sender;
    if (!rb.Checked)
    {
        return;
    }

    // Nur aktualisieren, wenn die Zusatz-Optionen sichtbar sind
    if (GbxZusatz.Visible)
    {
        LblAuswahl.Text = $"Auswahl: {rb.Text}";
    }
}
