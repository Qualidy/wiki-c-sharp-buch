    2.8 Schleifen und Steuerelemente

    ListBox füllen (1..n)

    Fülle eine ListBox (LstZahlen) mit den Zahlen 1 bis 10 per for. Wähle dann die 7 aus.

    Tipp

    Füge den Code in Form1_Load(...) hinzu.

    Lösung 
    public Form1()
{
    InitializeComponent();

    LstZahlen.Items.Clear();
    for (int i = 1; i <= 10; i++) LstZahlen.Items.Add(i);
    LstZahlen.SelectedItem = 7;
}
    Summe bilden

    Erstelle eine ListBox, eine TextBox und einen Button. Immer wenn auf den Button geklickt wird, wird die Zahl, die in der TextBox steht, in die ListBox eingetragen. Füge ein Label hinzu, dass die Summe der Elemente in der Listbox zeigt.

    Tipp

    Die Elemente in der Textbox sind string. Wenn du sie also aufsummieren willst, musst du sie vorher einen passenden Datentypen verwandeln.

    Lösung 
    // Button-Click-Handler: Zahl aus TextBox eintragen und Summe aktualisieren
private void BtnAdd_Click(object sender, EventArgs e)
{
    // Theoretisch müsste man hier prüfen, dass es sich bei der Eingabe
    // tatsächlich um eine Zahl handelt. Dies überspringen wir hier aber.
    LstZahlen.Items.Add(value);
    UpdateSumme();
    TxtZahl.Clear();

}

// Hilfsmethode: Summe der ListBox-Elemente berechnen und im Label anzeigen
private void UpdateSumme()
{
    int sum = 0;
    foreach (var item in LstZahlen.Items)
        sum += Convert.ToInt32(item);

    LblSumme.Text = $"Summe: {sum}";
}

// Optional: beim Laden initialisieren
private void Form1_Load(object sender, EventArgs e)
{
    InitializeComponent();

    LstZahlen.Items.Clear();
    UpdateSumme(); // zeigt "Summe: 0"
}
    Todo-App

    Erstelle mit deinen Bisherigen Mitteln eine Anwendung, um eine Todo-Liste zu zu erstellen. In einer ListBox sollen die Elemente aufgelistet sein. Es soll möglich sein neue Elemente hinzuzufügen und bestehende zu löschen.

    Lösung

    Die folgende Lösung wurde mit ChatGPT erstellt und enthält ein paar Funktionen, die über unsere Mittel hinausgehen. Diese sind aber verständlich oder gut kommentiert. Da wir hier also weitere Funktionen lernen können, lasse ich es als Lösung.

    // Fügt ein neues Todo aus der TextBox hinzu
private void BtnAdd_Click(object sender, EventArgs e)
{
    var text = TxtTodo.Text.Trim();
    if (string.IsNullOrEmpty(text))
    {
        MessageBox.Show("Bitte einen Text eingeben.");
        TxtTodo.Focus();
        return;
    }

    LstTodos.Items.Add(text);
    TxtTodo.Clear();
    TxtTodo.Focus();
}

// Löscht ausgewählte Todo(s) aus der ListBox
private void BtnDelete_Click(object sender, EventArgs e)
{
    if (LstTodos.SelectedIndices.Count == 0)
    {
        MessageBox.Show("Bitte mindestens ein Element auswählen.");
        return; // Die Methode wird vorzeitig beendet.
    }

    // Von hinten löschen, damit Indizes gültig bleiben
    for (int i = LstTodos.SelectedIndices.Count - 1; i >= 0; i--)
    {
        LstTodos.Items.RemoveAt(LstTodos.SelectedIndices[i]);
    }
}

// Optional: Initialisieren und Mehrfachauswahl erlauben
private void Form1_Load(object sender, EventArgs e)
{
    LstTodos.Items.Clear();
    LstTodos.SelectionMode = SelectionMode.MultiExtended; // Mehrfachauswahl zum Löschen
}

// Optional: Enter fügt hinzu, Entf löscht
private void TxtTodo_KeyDown(object sender, KeyEventArgs e)
{
    if (e.KeyCode == Keys.Enter)
    {
        e.SuppressKeyPress = true;
        BtnAdd.PerformClick();
    }
}

private void LstTodos_KeyDown(object sender, KeyEventArgs e)
{
    if (e.KeyCode == Keys.Delete)
    {
        BtnDelete.PerformClick();
    }
}

    Umständlicher Safe

    Erstelle vier ComboBoxen, jeweils mit den Zahlen von 0 bis 9.

    Speichere in einer Konstanten passwort einen String aus 4 Ziffern.

    Erstelle einen Button "Prüfe Code". Wenn dieser geklickt wird, werden die ausgewählten Ziffern der 4 ComboBoxen zu einem String zusammengesetzt und mit dem passwort verglichen. Wenn die Strings nicht übereinstimmen, erscheint in rot der Text "Falsches Passwort". Wenn sie übereinstimmen, erscheint ein Text mit einem Geheimnis von dir.

    Bonus: Beende das Programm, wenn drei Mal das falsche Passwort ausgewählt wurde.

    Lösung 
    // Konstante für das Passwort und Zähler für Fehlversuche
private const string passwort = "1234"; // 4-stellig
private int fehlversuche = 0;

// Beim Laden: ComboBoxen mit 0..9 füllen und initial auswählen
private void Form1_Load(object sender, EventArgs e)
{
    // Ohne Arrays: jede ComboBox einzeln füllen
    Cmb1.Items.Clear();
    for (int i = 0; i <= 9; i++)
    {
        Cmb1.Items.Add(i);
        Cmb2.Items.Add(i);
        Cmb3.Items.Add(i);
        Cmb4.Items.Add(i);
    }

    LblStatus.Text = string.Empty;
}

// Button-Handler: Code prüfen
private void BtnPruefe_Click(object sender, EventArgs e)
{
    string code = ($"{Cmb1.SelectedItem}{Cmb2.SelectedItem}{Cmb3.SelectedItem}{Cmb4.SelectedItem}");

    if (code == passwort)
    {
        LblStatus.ForeColor = Color.Green;
        LblStatus.Text = "Geheimnis: Ich habe ein süßes Gummienten-Maskottchen auf dem Schreibtisch.";

        // Optional: Nach Erfolg Eingaben deaktivieren
        Cmb1.Enabled = Cmb2.Enabled = Cmb3.Enabled = Cmb4.Enabled = false;
        BtnPruefe.Enabled = false;
        return;
    }

    // Falsches Passwort
    fehlversuche++;
    LblStatus.ForeColor = Color.Red;
    LblStatus.Text = "Falsches Passwort";

    if (fehlversuche >= 3)
    {
        Application.Exit(); // Bonus: nach 3 Fehlversuchen beenden
    }
}
