    Methoden – Signatur, Rückgabe, benannte/optionale Parameter

    Implementiere eine Methode PreisMitSteuer(decimal netto, decimal steuersatz = 0.19m), die den Bruttopreis zurückgibt. Rufe sie mit benannten Parametern auf und zeige die Ausgabe formatiert.

    Beispiel

    • PreisMitSteuer(netto: 59.90m)
    • PreisMitSteuer(netto: 59.90m, steuersatz: 0.07m)
    Lösung 
    private decimal PreisMitSteuer(decimal netto, decimal steuersatz = 0.19m)
{
    return netto * (1 + steuersatz);
}

private void CmdTest_Click(object sender, EventArgs e)
{
    decimal a = PreisMitSteuer(netto: 59.90m);
    decimal b = PreisMitSteuer(netto: 59.90m, steuersatz: 0.07m);
    LblAnzeige.Text = $"19%: {a:F2} €\n7%:  {b:F2} €";
}
    ref/out – Tauschen und Parsen

    Schreibe zwei kleine Helfer:

    • void Swap(ref int a, ref int b) vertauscht zwei Werte.
    • bool TryReadInt(string s, out int value) liefert true, wenn s eine Ganzzahl ist.

    Teste beides in einem Button‑Klick und zeige die Ergebnisse.

    Lösung 
    private void Swap(ref int a, ref int b) { (a, b) = (b, a); }

private bool TryReadInt(string s, out int value)
{
    return int.TryParse(s, out value);
}

private void CmdTest_Click(object sender, EventArgs e)
{
    int x = 3, y = 7;
    Swap(ref x, ref y);

    bool ok = TryReadInt(TxtEingabe.Text, out int v);
    LblAnzeige.Text = $"Swap: x={x}, y={y}\nParse: ok={ok}, v={v}";
}
    Rekursion – Fakultät und Fibonacci (kleine n)

    Implementiere zwei rekursive Methoden mit kleiner n‑Begrenzung:

    • long Fak(int n) mit Fak(0) = 1.
    • long Fib(int n) mit Fib(0)=0, Fib(1)=1.

    Zeige beide Werte für n aus einem NumericUpDown an und warne für zu großes n.

    Lösung 
    private long Fak(int n) => n <= 1 ? 1 : n * Fak(n - 1);

private long Fib(int n) => n <= 1 ? n : Fib(n - 1) + Fib(n - 2);

private void CmdCalc_Click(object sender, EventArgs e)
{
    int n = (int)NumN.Value; // z. B. Max 20
    if (n > 20)
    {
        MessageBox.Show("Bitte n ≤ 20 wählen.", "Warnung");
        return;
    }
    LblAnzeige.Text = $"Fak({n}) = {Fak(n)}\nFib({n}) = {Fib(n)}";
}
    Arrays – Minimum, Maximum, Durchschnitt

    Lies eine kommaseparierte Liste von Ganzzahlen aus TxtListe ein und berechne auf einem int[]:

    • Minimum und Maximum
    • Summe und Durchschnitt (auf zwei Nachkommastellen)

    Zeige die Ergebnisse mehrzeilig an.

    Lösung 
    private void CmdStats_Click(object sender, EventArgs e)
{
    var parts = TxtListe.Text.Split(',', StringSplitOptions.RemoveEmptyEntries);
    var arr = parts.Select(p => int.Parse(p.Trim())).ToArray();

    int min = arr.Min();
    int max = arr.Max();
    int sum = arr.Sum();
    double avg = arr.Average();

    LblAnzeige.Text = $"Min: {min}\nMax: {max}\nSumme: {sum}\nØ: {avg:F2}";
}
    Jagged Array – Tabellenförmig anzeigen

    Erzeuge ein gezacktes Array int[][], z. B. drei Zeilen mit unterschiedlicher Länge, fülle es mit Zahlen und gib jede Zeile in einer neuen Label‑Zeile aus.

    Hinweis

    • Verwende string.Join(" ", zeile) zum Formatieren.
    Lösung 
    private void CmdJagged_Click(object sender, EventArgs e)
{
    int[][] a =
    {
        new[] {1, 2, 3},
        new[] {4, 5},
        new[] {6, 7, 8, 9}
    };

    var lines = a.Select(row => string.Join(" ", row));
    LblAnzeige.Text = string.Join("\n", lines);
}
    Nullable und Null‑Koaleszenz

    Verwende int? für eine optionale Eingabe. Zeige den Wert an oder ersatzweise 0, wenn null. Nutze dazu den Null‑Koaleszenz‑Operator ??.

    Beispiel

    • int? vielleicht = null;
    • int sicher = vielleicht ?? 0;
    Lösung 
    private void CmdNullables_Click(object sender, EventArgs e)
{
    int? vielleicht = string.IsNullOrWhiteSpace(TxtEingabe.Text) ? null : int.Parse(TxtEingabe.Text);
    int sicher = vielleicht ?? 0;
    LblAnzeige.Text = $"Eingabe: {(vielleicht.HasValue ? vielleicht.Value.ToString() : "null")}\nErsatz: {sicher}";
}
    Tupel – Mehrfachrückgabe

    Schreibe static (int min, int max, double avg) Analyse(int[] a), die Minimum, Maximum und Durchschnitt eines Arrays liefert. Demonstriere den Aufruf mit Dekonstruktion.

    Lösung 
    private static (int min, int max, double avg) Analyse(int[] a)
{
    return (a.Min(), a.Max(), a.Average());
}

private void CmdAnalyse_Click(object sender, EventArgs e)
{
    int[] a = { 5, 1, 9, 3 };
    var (mi, ma, av) = Analyse(a);
    LblAnzeige.Text = $"Min: {mi}\nMax: {ma}\nØ: {av:F2}";
}
    Methodenüberladung – Drei Varianten Area

    Überlade eine Methode Area(...) für

    • Quadrat (side)
    • Rechteck (width, height)
    • Kreis (radius)

    und zeige jeweils die berechnete Fläche.

    Lösung 
    private double Area(double side) => side * side;
private double Area(double width, double height) => width * height;
private double AreaCircle(double radius) => Math.PI * radius * radius;

private void CmdArea_Click(object sender, EventArgs e)
{
    double a = Area(3);
    double b = Area(4, 2);
    double c = AreaCircle(2.5);
    LblAnzeige.Text = $"Quadrat: {a:F2}\nRechteck: {b:F2}\nKreis: {c:F2}";
}
