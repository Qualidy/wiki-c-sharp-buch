    WPF – Grid‑Layout mit zwei Spalten

    Erstelle ein WPF‑Fenster mit Grid und zwei Spalten.

    Layout

    • Linke Spalte fix 200 px, rechte Spalte *
    • Oben eine TextBox über beide Spalten
    • Unten zwei Buttons (links/rechts)
    Lösung 
    <Window xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="Layout" Height="300" Width="500">
  <Grid>
    <Grid.RowDefinitions>
      <RowDefinition Height="Auto"/>
      <RowDefinition Height="*"/>
    </Grid.RowDefinitions>
    <Grid.ColumnDefinitions>
      <ColumnDefinition Width="200"/>
      <ColumnDefinition Width="*"/>
    </Grid.ColumnDefinitions>

    <TextBox Grid.Row="0" Grid.ColumnSpan="2" Margin="8"/>

    <Button Grid.Row="1" Grid.Column="0" Margin="8" Content="Links"/>
    <Button Grid.Row="1" Grid.Column="1" Margin="8" Content="Rechts"/>
  </Grid>
</Window>
    WPF – Datenbindung an Slider

    Binde die FontSize eines TextBlocks an den Wert eines Sliders.

    Anforderungen

    • Slider Minimum 8, Maximum 48
    • Bidirektionale Bindung (Mode=TwoWay)
    Lösung 
    <Window xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="Binding" Height="200" Width="400">
  <StackPanel Margin="12">
    <Slider x:Name="Sizer" Minimum="8" Maximum="48" Value="16"/>
    <TextBlock Text="Hello WPF" FontSize="{Binding ElementName=Sizer, Path=Value, Mode=TwoWay}" />
  </StackPanel>
</Window>
    WPF – Commands mit Buttons

    Verbinde einen Button mit ApplicationCommands.Copy und reagiere im Code‑Behind.

    Lösung 
    <Window x:Class="Demo.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="Commands" Height="200" Width="300">
  <Window.CommandBindings>
    <CommandBinding Command="ApplicationCommands.Copy"
                    Executed="Copy_Executed"
                    CanExecute="Copy_CanExecute"/>
  </Window.CommandBindings>
  <StackPanel>
    <Button Command="ApplicationCommands.Copy" Content="Kopieren" Margin="8"/>
  </StackPanel>
</Window>

    private void Copy_CanExecute(object sender, CanExecuteRoutedEventArgs e) { e.CanExecute = true; }
private void Copy_Executed(object sender, ExecutedRoutedEventArgs e) { MessageBox.Show("Copy!"); }
    WPF – ItemsControl mit Datenliste

    Zeige eine Liste von Personen in einem ItemsControl mit einem DataTemplate an.

    Lösung 
    <Window x:Class="Demo.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="Items" Height="300" Width="400">
  <StackPanel>
    <ItemsControl x:Name="People">
      <ItemsControl.ItemTemplate>
        <DataTemplate>
          <StackPanel Orientation="Horizontal">
            <TextBlock Text="{Binding Name}" Margin="0,0,8,0"/>
            <TextBlock Text="{Binding Geburtsjahr}"/>
          </StackPanel>
        </DataTemplate>
      </ItemsControl.ItemTemplate>
    </ItemsControl>
  </StackPanel>
</Window>

    public partial class MainWindow : Window
{
    public MainWindow()
    {
        InitializeComponent();
        People.ItemsSource = new []
        {
            new Person("Alex",1992), new Person("Sam",1988)
        };
    }
}
    WPF – Animation mit Storyboard

    Animiere die Opacity eines Buttons zwischen 0.2 und 1.0 im Ping‑Pong‑Modus.

    Lösung 
    <Window xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="Anim" Height="200" Width="300">
  <Grid>
    <Button x:Name="Blink" Content="Blink" Width="120" Height="40">
      <Button.Triggers>
        <EventTrigger RoutedEvent="Loaded">
          <BeginStoryboard>
            <Storyboard AutoReverse="True" RepeatBehavior="Forever">
              <DoubleAnimation Storyboard.TargetProperty="Opacity"
                               From="0.2" To="1.0" Duration="0:0:1"/>
            </Storyboard>
          </BeginStoryboard>
        </EventTrigger>
      </Button.Triggers>
    </Button>
  </Grid>
</Window>
    WPF – Navigation zwischen Seiten

    Verwende ein Frame, um zwischen zwei Page‑Seiten zu navigieren.

    Hinweis

    • Lade die erste Seite im Loaded‑Ereignis.
    • Navigiere per Button‑Klick zur zweiten Seite.
    Lösung 
    <Window x:Class="Demo.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="Nav" Height="300" Width="400" Loaded="Window_Loaded">
  <DockPanel>
    <StackPanel DockPanel.Dock="Top" Orientation="Horizontal">
      <Button Content="Seite 1" Click="Btn1_Click" Margin="4"/>
      <Button Content="Seite 2" Click="Btn2_Click" Margin="4"/>
    </StackPanel>
    <Frame x:Name="MainFrame"/>
  </DockPanel>
</Window>

    private void Window_Loaded(object sender, RoutedEventArgs e) => MainFrame.Navigate(new Page1());
private void Btn1_Click(object s, RoutedEventArgs e) => MainFrame.Navigate(new Page1());
private void Btn2_Click(object s, RoutedEventArgs e) => MainFrame.Navigate(new Page2());
